### 024：see 同源词

see： 表示看见

sight：
1. 视力，视觉
2. 名胜，经典
```
（视力，视觉）：
Anne's sight is very good for someone of her age. 安妮的视力对于她这个年龄的人来说算非常好了。

（名胜，经典）：
In the afternoon, you'll have a chance to relax or see the sights. 大家下午可以休息，也可以去参观名胜
```

sightseeing：观光，游览（sight 派生词）
```
She swam and sunbathed, went sightseeing and relaxed. 她游泳，晒日光浴，去观光，然后休息
```

eyesight：视力
```
have good bad eyesight 有良好的视力
```

overlook
1. 俯视；眺望（从上往下看）
2. 忽略，未注意到（小，记忆点：因为从上向下看，所以会忽视，相当于 miss 也可以替换 miss）

```
（俯视；眺望） 
a restaurant overlooking the lake 俯视湖边的餐厅
Our back yard is overlook by several houses.好几栋房子都看得见（俯视）我家的后院

（忽略，未注意到）
He seems to have overlooked six important fact. 他好像忽略了六个重要的事实
```

### 025：man 同源词

women：女子
```
clothe for men 男人的服装
all men must die 人皆有意思

a single woman 单身女子
```

business：商业，买卖，生意，业务
记忆点：buys（忙碌） -> business，商业通常很忙碌
```
we do business with a number of italian companies 我们和一些意大利公司做生意
```

记忆点：商业上的男人 = 商人

chairwoman：女性负责人
记忆点：chair = 椅子，坐在椅上在的男人 = 主席
```
Potts ws appointed chairman of the education committee.  / Potts 当选为教育委员会主席
```

policeman：警察（男）
policewoman：警察（女）
记忆点：plice = 警察👮🏻

salesman：售货员，推销员（男）
salesman：售货员，推销员（女）
记忆点：sale = 卖，销售

### 026：marry 同源词
marry：与（某人）结婚，嫁或娶（某人）
```
They married young. 他们年级轻轻就结婚了

she didn't marry until she over fifty. 她直到五十多岁才结婚
```

marriage；
记忆点：age = 名词，名词的结婚和婚姻 = marriage
```
an offer of marriage （提供婚姻） = 求婚

After ten years of marriage, they are divorcing. 经过十年的婚姻，他们正在闹离婚
```

wed：结婚，娶，嫁
记忆点：结婚的具体动作，礼仪
```
The couple plan to wed next summer. 这两人计划在夏天结婚
```

wedding：婚礼；结婚庆典
记忆定：名词的婚礼
```
a wedding present  结婚礼物

Have you been invited to their wedding ? 他们有没有邀请你参加婚礼 ？
```

bride：新娘
记忆点：pride（骄傲） = bride （新娘），新娘就是我的骄傲
```
He introduced his new bride. 他介绍了他的新娘。
```

bridgeroom：新郎，刚刚结婚的男子
记忆点：groom，洗马的家伙

### 027：memory 同源词
词根：memo -> 记忆

memory：记忆力，记性
```
have a good memory 记忆力好
```

memorial：纪念碑，纪念章，纪念仪式
记忆点：记忆英雄
```
erect a war memorial 竖立战争纪念碑
This statur is a memorial to a great statesman. 这尊雕像是纪念一位伟大的政治家
```

memorize：记住，熟记
记忆点：reminder = 向外想起，memorize = 由内向外

remember：记得
```
remember doing sth 记得做过某事
I hope he remembers the wine. 我希望他记得带酒
Remember to take your P.E. clothes to school. 别忘了把你的运动服带到学校去
```

### 028：middle 同源词
middle：中间，中央，中部，当中
```
we rowed out towards the middle of the lake.
我们朝着湖中央划去
```

midday：中午，正午
记忆点：一天的中间
```
i'm meeting him at midday. 我中午要去见他
i got there around midday. 我是中午前后到那儿的
```

midnight：午夜，半夜 12 点钟
记忆点：晚上的 12 点

immediate：立即，即可的
记忆点：im med = 没有中间，没有间隔
```
i want an immediate reply 我想要立刻答复我
take immediate action 立刻采取行动
```

immediately：(adv) 
1. 立刻，马上，当即，毫不耽搁
2. 最接近地，直接地（在时空上）
```
She answered almost immediately. 她几乎当下就答复了
in the years immediately after the war 战后最初的几年
```

media：新闻媒体，传媒
```
the news medis 新闻媒体
```

medium
1. 传播媒介 -> media
2. 中等的，中号的

```
（传播媒介）Advertishing is a powerful medium. 广告是具有强大影响力的传播媒介
（中等的）What size shirt does he wear -- small, medium or large ? 他穿多大的衬衫 ？ 小号，中号，还是大号 ？
```

means：方法，手段，工具
记忆点：做某事的中间媒体
```
means of transport 交通工具
means of communication 通讯手段
```

meanwhile：期间，中间
```
i went to college, Meanwhile, all my friends go well-paid jobs. 我上大学那会，那是我的朋友全都找到了收入不错的工作
```

mean：意指某事，意思是
```
A dictionary tells you what words mean 词典是解释词义的工具书
What does this sentence mean ? 这句子是什么意思 ？
```

meaning（名词）：意思，意义，含义 
```
You can't say that these sounds have no meaning. 不能说这些声音全无意义
```
