# Docker 简介

为什么会有 Docker  ？

容器有很长的历史，但传统容器复杂，不易安装，管理和自动化也很困难。Docker 让容器变的简单



Docker 提供了什么 ？

* 提供一个简单，轻量的创建容器的方式
* 职责分离：解决 **程序在我这里运行还是好的** 类似扯皮的问题
* 面向微服务架构：容器只运行一个进程，天生的支持分布式



Docker 的核心组件：

* Docker 客户端和服务器
* Docker 镜像
* Registry
* Docker 容器



Registry 的两种类型：

* 公共 Registry：也叫 Docker Hub，由 Docker 公司运营，已有上万活跃用户和公共镜像
* 私有 Registry：企业内部自己架设的私有 Registry
